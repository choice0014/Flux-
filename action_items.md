# 권장 작업(요약)

1. 빌드 자동화: MSVC 빌드 스크립트(build.bat)는 존재하지만 CI가 없음. GitHub Actions 기반 빌드 워크플로우 추가 권장.
2. 불필요한 바이너리/IDE 파일(.vs/, *.obj, *.exe)을 리포지터리에서 제거하고 .gitignore에 추가 권장.
3. LICENSE 파일 추가 권장 (오픈소스 사용 시 명시 필요).
4. 민감정보 탐지 결과 검토 및 노출된 경우 즉시 교체/무효화 권장.
5. 테스트 스위트 추가: 현재 테스트 프레임워크가 명시되지 않음. 유닛 테스트 구조 도입 권장.
